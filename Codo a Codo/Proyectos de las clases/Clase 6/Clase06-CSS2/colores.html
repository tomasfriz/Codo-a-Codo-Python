Formas de determinar colores usando CSS

Por su nombre

Podemos referirnos a los colores por su nombre (aqui está la lista de los 140 colores disponibles mediante este método)

Color Crimson

Color Red

Color SlateBlue

Color SeaGreen

Por su valor RGB (Red Green Blue o Rojo Verde Azul)

Este método utiliza valores comprendidos entre 0 y 255 para la intensidad de cada uno de los tres colores utilizados como base para determinar el color final: rojo, verde y azul (en ese orden).

Color rgb(140, 71, 7)

Color rgb(0, 4, 255)

Color rgb(170, 187, 51)

Color rgb(46, 139, 87)

Por su valor hexadecimal

Este método utiliza valores en base 16 (hexadecimal) comprendidos entre 00 y FF para la intensidad de cada uno de los tres colores utilizados como base para determinar el color final: rojo, verde y azul (en ese orden).

Color #8c4707

Color #0004ff

Color #AABB33

Color #2E8B57

Por su valor RGBA (rgb + transparencia o canal alfa)

Este método es similar al rgb, pero agregando un cuarto valor que va de 0 a 1 para indicar que el color es completamente transparente o completamente opaco.

Color rgba(255, 30, 0, 1)

Color rgba(255, 30, 0, 0.7)

Color rgba(255, 30, 0, 0.4)

Por su valor RGBA (rbb + transparencia o canal alfa en %)

Este método es similar al anterior, pero el cuarto valor va de 0% a 100% para indicar que el color es completamente transparente o completamente opaco.

Color rgba(100%, 20%, 0%, 100%)

Color rgba(100%, 20%, 0%, 80%)

Color rgba(100%, 20%, 0%, 60%)

Color rgba(100%, 20%, 0%, 40%)

Todo esto se puede hacer usando CSS en un archivo separado, usando la etiqueta STYLE, etc.
